这些 GIF 大多是原始 Math-To-Manim [`public/readme-showcase/`](https://github.com/HarleyCoops/Math-To-Manim/tree/main/public/readme-showcase) bundle 的本地副本。新的 M2M2 recovery renders 只有在内容验证后才会加入，因此这个重写版本同时保留艺术北极星和当前管线目标。

它们不是装饰。它们定义了 M2M2 最终应该达到的标准：

- 有电影感的暗色数学；
- 干净的解释性图形；
- 可读符号和分阶段揭示；
- 把几何、拓扑、概率、微积分、物理和 ML 渲染成运动；
- README 尺寸下仍能传达核心想法的短循环。

## 用这个展厅作为目标

当 M2M2 生成成功 MP4 后，只有在视觉检查 loop 后，才把最强的教学节拍提升到这个展厅。好的 showcase GIF 应该具备：

1. 清晰的教学瞬间；
2. README 尺寸下可读的文字；
3. 稳定配色且没有 broken frames；
4. 足够短，便于快速浏览；
5. 描述解释概念，而不只是复述文件名。

根 README 包含 palette-optimized GIF creation 的 FFmpeg recipe。
