ErdosUnitDistanceProblem：创建一段 3 分钟的电影感 3D Manim explainer，讲述 Erdős 平面 unit distance problem 上由 AI 发现的突破。使用 warm shell/off-white 背景，不要黑底；配色应优雅，包含 porcelain white、royal purple、deep mathematical blue 和 metallic gold。情绪应像进入一座数学大教堂：干净、明亮、精确、令人惊叹。视觉风格应史诗但可读：sweeping camera moves、slow zooms into formulas、zoom-outs into moving point configurations，以及 layered opacity，让观众始终知道该看哪里。

开场是广阔 off-white plane，其中漂浮着 points。相机滑向两个点，画出长度为 1 的发光 gold segment，然后 zoom out 揭示许多可能的 unit-distance edges。用 LaTeX 引入中心问题：`u(n)=max #{ {p_i,p_j}: ||p_i-p_j||=1 }`。Zoom into `u(n)`、`max` 和 norm condition `||p_i-p_j||=1`；每个 symbol 都应转化为 geometry：`u(n)` 成为 counter，`max` 成为竞争 point clouds，norm condition 成为在点之间吸附的 gold unit ruler。

展示历史 baseline。先动画展示一条线上 n 个 points：一条有 `n-1` 条 unit edges 的简单 chain。然后相机抬升到 3D，chain 展开为 square grid，pale plane 上的 white points，blue horizontal/vertical edges 亮起，暗示约 `2n` 个 unit distances。随后 rescale 并旋转 grid，使其成为更密的 black/blue network-like lattice，并用紫色显示公式 `n^{1+C/log log n}`。Zoom into exponent `C/log log n`，让它向 0 缩小，直观解释这只是 barely superlinear。屏幕上放置长期 belief：`u(n) <= n^{1+o(1)}`。让它像 grid 上方的 transparent glass ceiling。

现在创造突破。Glass ceiling 中出现 gold fracture，公式 `n^{1+delta}` 穿过裂缝上升，带 callout `delta>0` 和 `delta=0.014`。不要过度宣称：说明它对 infinitely many n 成立，并且 proof 构造的 configurations 具有 polynomially more unit-distance pairs，超过 conjectured grid paradigm 所允许的数量。使用 dramatic split-screen：左侧显示 `n^{1+o(1)}` 回落到接近 linear growth；右侧显示 `n^{1+delta}` 随 n 增长而拉开距离，gold 3D curve 升到 blue/purple reference plane 上方。

解释令人意外的桥梁。先用 Gaussian integers `a+bi` 作为 complex plane 中的 white-and-blue integer lattice。Points 在坐标 `a+bi` 处 pulse，unit differences 显示为 gold arrows。然后穿过公式 `a+bi`，进入发光的 abstract number-field chamber：rings of algebraic integers、factorization trees 和 symmetric purple/gold constellations。把更丰富 algebraic number fields 表示为 stacked translucent planes 或 nested 3D lattices，它们比 Gaussian integer grid 有更多 symmetries。提到 infinite class field towers 和 Golod-Shafarevich theory 作为深层 existence machinery，但视觉保持 conceptual：glowing field layers 的 towers 向上升起，edges 投影回 Euclidean plane，成为许多 unit-length differences。

主要 aha moment：这个 elementary plane problem 不是靠更努力地推进 square grid 解决的，而是通过从 algebraic number theory 导入 hidden structure 解决的。从 purple number-theory tower 到 blue geometric point cloud 动画架起一座桥。桥落地时，许多 gold unit edges 同时亮起，edge counter 加速超过 grid baseline。相机应反复 zoom into formulas 再 zoom out into motion：`u(n)`、`n^{1+o(1)}`、`n^{1+delta}`、`a+bi`、`delta>0` 和 `||p_i-p_j||=1` 都应成为可见的几何动作。

结尾是宽阔 3D tableau：旧 square grid 作为平静 blue foundation，新 construction 作为 gold-purple constellation 升在其上，一个小 AI reasoning node 出现，但不是魔法，而是像 searchlight 一样揭示 hidden bridge。以大号干净文字结束：“A simple distance question hid a number-theoretic universe.” 下方写：“AI found the path; mathematicians checked the proof; geometry now has a new bridge.” 保持缓慢电影 pacing，每个 reveal 后留 breathing room，避免拥挤。最终动画应像严谨教育 discovery trailer，而不是 slide deck。
